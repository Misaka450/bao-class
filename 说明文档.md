# 小学成绩管理系统 - 项目说明文档

## 项目概述
- **项目名称**: 小学成绩管理系统 (BaoClass)
- **技术栈**: React 19 + TypeScript + Cloudflare Serverless
- **AI集成**: DeepSeek-V3.2 (文本生成) + Qwen3-VL (OCR识别)
- **数据库**: Cloudflare D1 (SQLite) + KV + R2
