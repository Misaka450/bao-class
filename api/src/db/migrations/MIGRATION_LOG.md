## 核心表结构

### 用户系统
- `users` - 用户表（管理员、教师、学生、家长）

### 班级管理
- `classes` - 班级表
- `students` - 学生表
- `courses` - 科目表

### 考试系统
- `exams` - 考试表
- `exam_courses` - 考试 - 科目关联表（多对多）
- `scores` - 成绩表

### 教学功能
- `lesson_plans` - 备课表
- `homework` - 作业表
- `homework_submissions` - 作业提交表

### 系统功能
- `audit_logs` - 操作日志表
- `ai_comments` - AI 评语表

## 工具脚本

### 开发环境
- `reset.sql` - 完全重置数据库（删除所有表）
- `clear-data.sql` - 清空所有数据但保留表结构
- `insert-data.sql` - 导入测试数据

### 生产环境
- `schema.sql` - 完整数据库结构
- `indexes.sql` - 性能优化索引

## 注意事项

1. **迁移顺序**: 必须按照版本号顺序执行迁移
2. **数据备份**: 执行迁移前务必备份数据
3. **测试环境**: 所有迁移先在测试环境验证
4. **回滚方案**: 每个迁移都应该有回滚方案
